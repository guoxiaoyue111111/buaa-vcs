# 一级工程实践大作业

## 项目说明

### 结构

```
 |-module           # 子模块
   |-appliance      # 家用电器模块
   |-command        # 指令解析模块
   |-io             # 输入输出模块
   |-voice          # 语音模块
 |-const.py         # 字典，定义全局状态等元数据
 |-view.py          # 展示模块
 |-main.py          
```

### 执行
```
pip install -r requirements.txt -i https://pypi.douban.com/simple
python main.py
```
